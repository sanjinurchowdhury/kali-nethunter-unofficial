# kali-nethunter-unofficial
this is an rootless version of nethunter which is modified by sanjinurchowdhury. It uses GNOME desktop. environment by default. You have install other desktop environment manually. 
#Install rootless nethunter (unnoficial)
1.run 'git clone https://github.com/sanjinurchowdhury/kali-nethunter-unofficial.git'
2.run 'cd kali-nethunter-unofficial'
3.run 'chmod +x install-nethunter-termux.sh'
4.now execute './install-nethunter-termux.sh' and follow the steps

note that you have to install termux from fdroid and nethunter kex from nethunter store
